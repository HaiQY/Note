# 研究报告

## 一、引言
### 1.1 研究背景
笔记是学习过程中的重要载体。然而，纸质笔记随着时间推移容易丢失且难以检索。传统的数字化方式（如手动打字）耗时费力。如何利用人工智能技术自动处理手写笔记，并将其转化为可复习的知识卡片，是本研究的核心问题。

### 1.2 研究目的
本项目旨在构建一个端到端的笔记处理平台，实现“拍摄 - 识别 - 整理 - 复习”的闭环，帮助学生从繁琐的整理工作中解放出来，专注于知识内化。

## 二、系统设计与架构
### 2.1 系统架构
系统采用 B/S 架构，后端基于 Python FastAPI 框架，前端采用轻量化设计。整体分为数据接入层、业务逻辑层和数据持久层。
*   **数据接入层：** 负责图片上传、格式校验。
*   **业务逻辑层：** 包含 OCR 服务、结构分析、分类服务、关键词提取、AI 卡片生成。
*   **数据持久层：** 使用 SQLite 数据库存储笔记元数据、分类信息及复习记录。

### 2.2 功能模块
1.  **笔记上传与 OCR 识别：** 支持多种图片格式，自动倾斜矫正，返回置信度评分。
2.  **文档结构识别：** 识别表格、公式（LaTeX 格式）及图表，生成结构化 Markdown 文档。
3.  **自动分类与关键词：** 基于关键词匹配度自动归类，提取 Top-K 关键词。
4.  **AI 复习卡片：** 生成 Q&A、填空等多种题型，支持难度分级。
5.  **全文搜索：** 支持按标题、内容、关键词及时间范围筛选。

## 三、核心算法与实现
### 3.1 高精度 OCR 识别
采用 PaddleOCR v3.4.0 引擎。为提升手写体识别率，启用了角度分类器（use_angle_cls=True）进行倾斜矫正。对于识别结果，可选接入 GPT 模型进行语义校正，解决手写潦草导致的错别字问题。
*   **实现效果：** 在测试集上，清晰手写体的识别置信度平均分超过 0.92。

### 3.2 智能分类算法
采用基于关键词匹配的相似度计算。
*   **公式：** $\text{similarity} = \frac{\text{匹配关键词数}}{\text{分类总关键词数}}$
*   **流程：** 遍历所有预定义分类（如物理、数学、英语），计算笔记内容与分类关键词库的匹配度，选取最高分作为分类结果。若最高分低于阈值，则归为“其他”。

### 3.3 关键词提取策略
实现了三种算法供选择：
1.  **连续 TF-IDF：** 基于词频 - 逆文本频率，计算速度快，适合通用场景。
2.  **TextRank：** 基于图算法，适合长文本和学术文本，能捕捉词间关系。
3.  **Jieba 分词过滤：** 保留名词动词，去除停词，针对中文优化。
实验发现，对于课堂笔记片段，TF-IDF 表现最为稳定。

### 3.4 SM-2 间隔重复复习算法
复习卡片模块核心采用 SM-2 算法。
*   **参数：** 间隔 $I$，易度因子 $E_F$，用户评分 $q$。
*   **更新逻辑：**
    *   $I(1) = 1, I(2) = 3$
    *   $I(n) = I(n-1) \times E_F$
    *   $E_F$ 根据用户评分动态调整（0-5 分）。
*   **效果：** 能够根据用户掌握情况，自动延长已掌握知识的复习间隔，缩短未掌握知识的复习间隔。

## 四、研究过程与测试结果
### 4.1 测试环境
*   **硬件：** 本地单机运行，内存 16GB，支持 GPU 加速。
*   **软件：** Python 3.12, FastAPI 0.131.0, SQLite。

### 4.2 功能测试
1.  **OCR 识别测试：** 上传 50 张不同字迹的笔记图片，置信度低于 0.9 的占比约 8%，主要通过 AI 优化模块修正。
2.  **分类准确性：** 在预设 7 个分类类别下，自动分类准确率约为 89%。
3.  **卡片生成：** 成功生成选择题、填空题等多种类型卡片，内容相关性高。

## 五、创新点与研究价值
### 5.1 创新点
1.  **结构化识别：** 不仅识别文本，还能识别笔记中的表格和公式，并转换为 Markdown/LaTeX 格式。
2.  **闭环复习：** 将笔记整理与间隔重复复习算法结合，实现了从“记录”到“内化”的闭环。
3.  **混合关键词提取：** 结合 TF-IDF 与 TextRank，适应不同长度的笔记内容。

### 5.2 研究价值
本项目证明了利用现有 AI 技术低成本构建个性化学习工具的可行性，减少了学生机械性整理时间.

## 六、结论与展望
### 6.1 结论
本研究成功开发了一套手写笔记智能整理系统，实现了预期目标。系统能够有效识别手写内容，自动分类整理，并生成科学的复习计划。测试结果表明，该系统具有较高的实用价值。

### 6.2 不足与展望
1.  **复杂公式识别：** 对于极其复杂的手写数学公式，识别率仍有提升空间。
2.  **多用户支持：** 当前版本主要为单机部署，未来可扩展为云端多用户服务。
3.  **个性化模型：** 未来可尝试微调 OCR 模型以适应特定用户的字迹。
