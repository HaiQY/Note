# 研究方案

## 背景

从学习过程中，我们发现课堂笔记存在内容杂乱、信息分散等问题，容易遗忘和对一些知识点模糊不清。且知识需要一些科学的方法（如关键词、分类、系统梳理总结）才能更好地理解。由此，为了帮助学生提升学习效率和效果，我们产生了开发该软件的想法。

## 目标

1. 开发一款适用于学生群体的智能笔记整理软件；
2. 通过方便的图片上传来录入笔记；
3. 实现笔记的自动分类、关键词提取、内容检索等功能；
4. 实现笔记与复习计划（「卡片」）的联动，帮助学生回顾复习。

## 方法

 - 访谈调查：收集学生用户的痛点与需求；
 - 实验开发：通过编程实现软件功能，根据实际体验优化；
 - 用户测试：广泛选取多种样本进行试用，评估软件的实用性与用户体验，迭代优化。
