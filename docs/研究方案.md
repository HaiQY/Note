# 研究方案

## 一、选题缘由与研究背景
### 1.1 研究背景
在当前的学习过程中，学生依然大量依赖纸质手写笔记。手写笔记具有灵活性高、利于记忆等优点，但也存在难以检索、不易整理、复习效率低等痛点。随着人工智能技术的发展，光学字符识别（OCR）和自然语言处理（NLP）技术已趋于成熟，为手写笔记的数字化和智能化处理提供了技术基础。

### 1.2 问题陈述
目前市面上的笔记应用多侧重于电子笔记的记录，针对“纸质笔记数字化后如何高效管理”的解决方案较少。主要存在以下问题：
1.  **整理困难：** 数字化后的文本缺乏自动分类和标签，形成“数据孤岛”。
2.  **复习低效：** 缺乏科学的复习计划引导，难以将笔记内容转化为有效的复习材料。

### 1.3 研究意义
本项目旨在开发一套端到端的手写笔记智能处理平台，实现从图片拍摄到知识管理的全流程自动化。通过 AI 技术辅助学生整理笔记，并利用间隔重复算法生成复习卡片，提高学生的学习效率和知识留存率。

## 二、研究目标
1.  **构建高精度 OCR 识别模块：** 实现对手写笔记图片的高精度文本提取，支持倾斜矫正。
2.  **实现智能分类与关键词提取：** 基于文本内容自动将笔记归类，并提取核心关键词。
3.  **开发 AI 驱动复习系统：** 利用大语言模型生成复习卡片，并结合 SM-2 算法规划复习路径。
4.  **完成系统原型开发：** 形成一套可运行的软件系统，包含上传、识别、管理、复习功能。

## 三、研究内容
### 3.1 笔记数字化与结构识别
研究如何利用深度学习模型（如 PaddleOCR）识别手写文本，并利用文档结构分析技术（PPStructure）识别笔记中的表格、公式和图表，将其转换为 Markdown 格式。

### 3.2 基于关键词的自动分类算法
研究基于关键词匹配度的文本分类算法。通过构建分类候选项及其关键词库，计算笔记内容与分类关键词的相似度，实现笔记的自动归档。

### 3.3 关键词提取技术研究
对比 TF-IDF 算法与 TextRank 算法在中文笔记场景下的表现，研究如何结合分词技术（Jieba）和停词过滤，准确提取笔记的核心知识点。

### 3.4 间隔重复复习算法应用
研究 SM-2 间隔重复算法在教育场景的应用。根据用户对复习卡片的评分（掌握程度），动态调整下次复习时间，实现个性化复习计划。

## 四、技术路线与研究方法
### 4.1 技术栈选择
*   **后端框架：** Python FastAPI。
*   **OCR 引擎：** PaddleOCR v3.4.0，支持中文手写体识别及角度分类。
*   **AI 模型：** 接入大语言模型 API 用于内容校正和卡片生成。
*   **数据库：** SQLite，用于存储笔记元数据、卡片信息及复习记录。

### 4.2 算法原理
1.  **分类算法：** 采用相似度计算公式 $Similarity = \frac{\text{匹配关键词数}}{\text{分类总关键词数}}$，选取匹配度最高的分类。
2.  **复习算法：** 采用 SM-2 算法，根据用户评分 $q (0-5)$ 更新易度因子 $E_F$ 和复习间隔 $I$。
    *   公式：$I(n) = I(n-1) \times E_F$
    *   公式：$E_F = E_F + (0.1 - (5-q) \times (0.08 + (5-q) \times 0.02))$

### 4.3 研究方法
*   **文献研究法：** 查阅 OCR 技术及间隔重复理论相关文献。
*   **实验法：** 收集不同字迹的笔记图片进行测试，优化参数。
*   **行动研究法：** 使用广泛的样本进行测试，不断调整优化。

## 六、可行性分析
*   **技术可行性：** 所选用的 PaddleOCR 和 FastAPI 均为开源成熟技术，文档齐全。
*   **操作可行性：** 系统流程设计简洁，用户上传即可自动处理，符合学生使用习惯。
*   **经济可行性：** 主要依赖开源模型和本地算力，成本低廉。
